# Global Media & Streaming Trends Dashboard

## Project Objective
This project was designed as a dedicated study in **Exploratory Data Analysis (EDA)** and **Data Visualization**. The primary goal was to take a raw dataset of 9,600+ titles and identify global trends in content production, rating distributions, and genre popularity.

## Analysis Workflow
1.  **Data Cleaning:** Handled missing values for directors and standardized genre categories across 519 unique types.
2.  **Metric Definition:** Established core KPIs including Total Titles, Director Count, and Content Age (1920–2021).
3.  **Visual Hierarchy:** Designed a dark-themed dashboard to prioritize high-level stats while allowing for granular geographical and temporal filtering.

## Key Insights
* **The Streaming Boom:** Visual evidence of the exponential growth in TV content vs. Movies starting around 2015.
* **Global Hubs:** Identified the US and India as the primary drivers of content volume in this dataset.
* **Content Maturity:** Analysis shows a heavy concentration in '13+' and '16+' ratings, indicating a target demographic shift toward young adults.

## Tools Used
* **Visualization:** Power BI / Tableau
* **Design:** High-contrast dark UI for professional readability.
